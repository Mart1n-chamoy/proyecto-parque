# 🎨 Frontend React - Proyecto-Parque

Interfaz moderna para el sistema de cobranza automatizada con IA.

## 🚀 Quick Start

### Instalación

```bash
cd frontend
npm install
```

### Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

### Build para Producción

```bash
npm run build
npm run preview
```

## 📁 Estructura de Carpetas

```
frontend/
├── src/
│   ├── api/              # Servicios de API
│   ├── components/       # Componentes React
│   ├── pages/            # Páginas principales
│   ├── store/            # Estado global (Zustand)
│   ├── styles/           # Estilos CSS
│   ├── App.jsx
│   └── main.jsx
├── package.json
├── vite.config.js
└── index.html
```

## 🛠️ Tecnologías

- React 18 + Vite
- React Router
- Zustand (State Management)
- Axios (HTTP Client)
- TailwindCSS

## 📊 Páginas

- `/dashboard` - Panel principal
- `/clients` - Gestión de clientes
- `/batches` - Gestión de lotes

## 🔌 API Integration

Servicio completo en `src/api/client.js` con endpoints para:
- Clientes
- Llamadas
- Lotes

## 🎨 Estilos

TailwindCSS + Lucide Icons
